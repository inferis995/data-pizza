Clone Repository

Terminal
# Clone the repository
git clone https://github.com/inferis995/data-pizza.git
cd data-pizza
2

Install Dependencies

Terminal
# Core Datapizza AI framework
pip install datapizza-ai

# Official tools
pip install datapizza-ai-tools-sqldatabase
pip install datapizza-ai-tools-duckduckgo
pip install datapizza-ai-tools-filesystem
pip install datapizza-ai-tools-web-fetch

# Vector stores for RAG
pip install qdrant-client chromadb
3

Configure Claude Code

Add Data Pizza skills to your Claude Code configuration:

claude_config.json
{
  "skills": [
    {
      "name": "datapizza-agent-creator",
      "path": "C:\\\\Users\\\\YourUser\\\\data-pizza\\\\skills\\\\datapizza-agent-creator\\\\SKILL.md"
    },
    {
      "name": "datapizza-tool-builder",
      "path": "C:\\\\Users\\\\YourUser\\\\data-pizza\\\\skills\\\\datapizza-tool-builder\\\\SKILL.md"
    }
  ]
}
4

Set API Keys

Terminal
export OPENAI_API_KEY="your-openai-key"
# or
export ANTHROPIC_API_KEY="your-anthropic-key"

Code Examples

See Data Pizza in action with these examples

Customer Service Agent

Generated with Agent Creator skill

from datapizza.agents import Agent
from datapizza.clients.openai import OpenAIClient
from datapizza.tools import tool

@tool
def get_customer_info(customer_id: str) -> str:
    """Get customer information."""
    return fetch_customer(customer_id)

client = OpenAIClient(api_key="YOUR_KEY")
agent = Agent(
    name="customer_service",
    system_prompt="Help customers with their inquiries.",
    client=client,
    tools=[get_customer_info]
)

Database Tool

Generated with Tool Builder skill

from datapizza.tools import tool
from datapizza.tools.SQLDatabase import SQLDatabase

db = SQLDatabase(db_uri="sqlite:///data.db")

@tool
def get_customer_data(customer_id: str) -> str:
    """Retrieve customer data safely."""
    try:
        query = f"SELECT * FROM customers WHERE id = {customer_id}"
        result = db.run_sql_query(query)
        return format_customer(result)
    except Exception as e:
        return f"Error: {str(e)}"
